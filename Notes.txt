## First project: 
https://www.w3schools.com/django/django_create_project.php
```
django-admin startproject my_tennis_club
python manage.py runserver
python manage.py startapp members
```

Important: When DEBUG = False, Django requires you to specify the hosts you will allow this Django project to run from.

In production, in the settings.py file, this should be replaced with a proper domain name:

ALLOWED_HOSTS = ['yourdomain.com']
